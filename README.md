# Blue Witness Labeler App


1. Grab a tweet
2. Get user's ranking of said tweet
3. Store tweet and rank in DB -> row(tweets, labels)
4. Train Bert with new data!


### Tech Stack
- Logic: Python
- API: Flask
- Interface: HTML
- Styling: CSS
- Graphs: Plotly
- Hosting: AWS Elastic Beanstalk
- Database: AWS RDS PostgreSQL
